Project Two

  • BRANDING & DESIGN

  • SEPTEMBER 2020

"In photography there is a reality so subtle that it becomes more real than reality.” A picture isn’t always worth a thousand words but Photography Blogs provide context for the picture. So this project of ours is for the same. For designing this website we used HTML5, CSS3 and Bootstrap as a User interface. We added JavaScript for several captivating effects used on this website. For showing alert or success messages we have took advantage of Flash Library. Our Backend on this website consists of a server, an application, and a database.

We used NodeJs which is a runtime environment that helped us to cover various aspects on it. MongoDB is a Non- relational database which we attached that enabled us to store data in a very convenient and effective way. We have used the Passport library for User Authentication. For further more security, we enabled reCaptcha for prevention of bot from our website. For searching particular blogs, we added a fuzzy search and have provided a Rating and Comment facility to each Blog so that users can rate and give their views on it. We inculcated these features to make it appealing for a huge potential audience. Our website will allow this blog to express out in ways that a photo can’t do alone.
